# 課題05 -API活用連携アプリ-

## ①課題内容（どんな作品か）
- トランプを次々に引いて、数の総和が100を超えたら負けになるゲーム。
- 数の総和の検知をコレクションを使って行っているので、複数人で遊ぶことが可能。

## ②工夫した点・こだわった点

- 対戦型ゲームにしていること。
- アプリのデプロイを行っていること。
